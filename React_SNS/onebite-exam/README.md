# 도구

- React + TypeScript + Supabase
- 컴포넌트 스타일링 TailwindCss
- 전역 상태 관리 Zustand
- 서버 상태 관리 Tanstack Query

# 기능

- 사용자 인증/ 인가 기능
  : 이메일 로그인 및 회원가입
  : GitHub 소셜 로그인
  : 비밀번호 찾기
  : 세션 유지
  : 라우트 가드

- 무한 스크롤 기능
  : useInfinite Query 사용
  : 로딩 상태 처리
  : 캐시 데이터를 활용한 데이터 요청 최적화

- 이미지 업로드 기능
  : Supabase 이미지 스토리지 개념
  : 버킷 설정 및 이미지 업로드 & 삭제

- 포스트 좋아요 기능
  : 좋아요 중복 방지 with RPC(원격 프로시저 호출)
  : 낙관적 업데이트를 통한 빠른 결과 반영

- 무한 중첩 댓글 기능
  : 댓글 생성/수정/삭제 기능
  : 무한 중첩 댓글 기능
  : 재귀적 컴포넌트 렌더링

- 테마 변경 기능
  : 다크 모드 & 라이트 모드
  : 로컬스토리지를 이용한 테마 유지
